> Task for KoreAi

## Clone the project

```sh
git clone https://github.com/SubashMourougayane/koreAiTask.git

cd koreAiTask
```

## Install

```sh
npm install
```

## Create .env file

```sh
copy the sample env file to .env file and add values to it
```

## Test Server

```sh
npm run test
```

## Start Server

```sh
npm start
```

## Tail Logs

```sh
npm run logs
```

## Stop Server

```sh
npm run stop
```
